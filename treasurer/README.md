# Agentic AI Treasurer

> An AI-powered personal crypto treasurer built for Web3 users — tracking wallets, analyzing transactions, and delivering intelligent financial insights.


## Product Overview

### What is Agentic AI Treasurer?

Agentic AI Treasurer is a decentralized application (dApp) that functions as an intelligent financial assistant for Web3 users. It connects directly to a user's crypto wallet on the HeLa blockchain and provides real-time visibility into their financial activity — balances, transactions, spending patterns, and budget health — all processed and interpreted by an AI agent that delivers actionable, personalized recommendations.

Unlike traditional finance apps that work with bank accounts or centralized data, this dApp operates entirely on-chain. It reads wallet data, interprets it through an AI layer, and presents the user with clear, human-readable financial insights — no spreadsheets, no manual tracking, and no financial expertise required.

### What Problem Are We Solving?

Web3 users today face a significant gap: they have complete ownership of their funds, but almost no intelligent tooling to understand or manage them. Most wallets show a list of transactions with no analysis, no categorization, and no guidance. Users are left to manually piece together their own financial picture.

This project solves that problem by acting as an autonomous financial layer on top of any connected wallet — transforming raw on-chain data into structured financial intelligence.

## Use Case

### Who Is This Built For?

Agentic AI Treasurer is designed for:

- **Everyday Web3 users** who hold and transact in crypto regularly and want to understand where their funds are going
- **DeFi participants** who interact with multiple protocols and need consolidated financial oversight
- **Crypto-native individuals** transitioning from traditional finance who are accustomed to budgeting tools and want equivalent functionality in Web3
- **Developers and early adopters** on the HeLa network looking for practical, utility-focused dApps

### Why Does This Matter for Users?

- Users gain a clear picture of their on-chain financial behavior without needing to export data or rely on third-party analytics platforms
- The AI layer removes the burden of manual interpretation — it reads the data and explains what it means in plain language
- Budget insights and spending pattern detection empower users to make smarter decisions with their crypto holdings
- Built natively on HeLa, the dApp is fast, low-cost, and accessible to the growing HeLa ecosystem

## Features

- **Wallet Integration** — Connect seamlessly via MetaMask to fetch live on-chain data
- **Balance Tracking** — Real-time display of wallet balances on the HeLa blockchain
- **Transaction Analysis** — Detailed breakdown and history of all wallet transactions
- **Expense Monitoring** — Categorized tracking of outflows and spending behavior
- **Budget Insights** — Visual summaries of financial activity over time
- **AI Financial Suggestions** — Intelligent, context-aware recommendations generated by the AI agent
- **Responsive UI** — Clean, modern interface built with Tailwind CSS and React.js

## Architecture

### How Does the Product Work?

The application is structured as a three-layer system: a React.js frontend, a Python FastAPI backend with an embedded AI agent, and a HeLa blockchain integration layer.

```
User (Browser)
      |
      v
React.js Frontend  (Vite + Tailwind CSS)
      |
      |-- MetaMask Wallet Connection
      |-- Sends wallet address to backend
      |
      v
FastAPI Backend  (Python)
      |
      |-- Blockchain Data Fetcher  (HeLa RPC)
      |-- Transaction Parser and Categorizer
      |-- AI Agent  (Financial Analysis Engine)
      |-- REST API Endpoints
      |
      v
HeLa Blockchain
      |
      |-- On-chain Wallet Data
      |-- Transaction History
      |-- Smart Contract Interactions
```


## HeLa Integration

### How Is This dApp Leveraging the HeLa Network?

Agentic AI Treasurer is built natively on the **HeLa blockchain** and leverages its infrastructure in the following ways:

- **Wallet Connectivity** — MetaMask is configured to connect directly to HeLa, enabling users to interact with their HeLa-based wallets without bridging or wrapping assets
- **On-Chain Data Fetching** — The backend connects to HeLa's RPC endpoint to read wallet balances and full transaction histories in real time
- **Smart Contract Deployment** — Contracts are deployed on HeLa using Hardhat, enabling structured on-chain data retrieval and future extensibility for financial logic
- **Low-Cost Transactions** — HeLa's fee structure makes it practical for frequent data interactions and micro-transaction analysis that would be cost-prohibitive on higher-fee networks
- **EVM Compatibility** — HeLa's EVM compatibility allows the use of standard Solidity contracts and Web3 tooling (Hardhat, ethers.js) without custom modifications
